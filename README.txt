A shortcode that displays a sortable table of users filtered by role - viewable by admin only.

== Installation ==

1. Copy the folder "users-role-shortcode" into the /wp-content/plugins/ folder
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Add `[users-role-table]` shortcode to a post

== Description ==
A shortcode that displays a sortable table of users filtered by role. This is a sample project.

**Example:**
`[users-role-table]`
